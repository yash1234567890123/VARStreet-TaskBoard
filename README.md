# Task Management System (VARStreet Technical Assessment)

A full-stack task management application built with .NET 8 and React.

##  Features
- **Backend:** C# API with SQLite, EF Core, and Pagination.
- **Frontend:** React (Vite) with Tailwind CSS.
- **Data Integrity:** Implements the required Project-Task-Comment hierarchy with strict Enums for Status and Priority.

##  Setup Instructions
1. **Backend:** Navigate to `/TaskBoard.Api` run dotnet restore
   and `dotnet run`.
2. **Frontend:** Navigate to `/task-board-ui`, run `npm install` and `npm run dev`.
